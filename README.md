## :pencil: Sobre o projeto

Uma rede social similar ao Twitter com funcionalidades de posts, chat e stories, criada utilizando a stack MERN. O objetivo do desenvolvimento é praticar o desenvolvimento de testes e seguir o conceito de Mobile First, além de adquirir ainda mais experiência com as tecnologias utilizadas.


## :computer: Principais Tecnologias utilizadas
- [TypeScript](https://www.typescriptlang.org/)
- [React](https://react.dev/) 
- [Sass](https://sass-lang.com/)
- [Axios](https://axios-http.com/ptbr/)
- [Node](https://nodejs.org/en)
- [Express](https://expressjs.com/)
- [MongoDB](https://www.mongodb.com/)
- [Jest](https://jestjs.io/pt-BR/)
